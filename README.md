# circleci_elixir

Teamweek's Circle CI Docker Image for Elixir

## Intro

- Circle CI's Elixir + Node docker image. (circleci/elixir:1.6)
- wkhtmltopdf
- libs for deploy

## Build

`docker build -t teamweek/circleci_elixir:latest .`
